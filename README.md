# EFT Ingeniería de Software - Edificio El Mirador

Proyecto desarrollado para la Evaluación Final Transversal de Ingeniería de Software.

## Descripción del proyecto

Este proyecto propone un sistema web para la gestión de gastos comunes del Edificio El Mirador, orientado a mejorar la administración de pagos, morosidad, informes, solicitudes y comunicación con los residentes.

## Problema identificado

La administración del edificio presenta dificultades para gestionar pagos de gastos comunes, controlar morosidad, generar informes y entregar información transparente a propietarios, arrendatarios y residentes.

## Objetivo general

Desarrollar una solución tecnológica que permita automatizar la gestión de gastos comunes, registrar pagos, controlar morosidad, enviar recordatorios y entregar información clara a la administración y residentes.

## Producto Mínimo Viable

El prototipo incluye:

- Inicio de sesión para administrador y residente.
- Dashboard del administrador.
- Gestión de residentes.
- Gestión de pagos.
- Control de morosidad.
- Envío de recordatorios.
- Informes de recaudación y pagos.
- Solicitudes y quejas de residentes.
- Panel del residente.
